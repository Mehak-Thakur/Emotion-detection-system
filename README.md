# Emotion Detection System

Deep Learning based Facial Emotion Recognition using ConvNeXtV2.

## Streamlit Deployment link: https://emotion-detection-system-mehak.streamlit.app/

## Dataset
RAF-DB

## Model
ConvNeXtV2 Base

## Features

- Upload Image
- Emotion Prediction
- Confidence Score
- Probability Graph

## Classes

- Angry
- Disgust
- Fear
- Happy
- Neutral
- Sad
- Surprise
